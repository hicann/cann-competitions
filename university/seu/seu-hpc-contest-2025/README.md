# 东南大学高性能计算竞赛 2025

## 🏆 赛事介绍
东南大学高性能计算竞赛是由东南大学计算机科学与工程学院主办，面向全校学生的高水平 HPC 竞赛。赛事聚焦于 CANN 平台下的高性能计算优化，旨在培养学生的并行计算能力和性能优化技巧。

## 📅 赛事时间
- **报名时间**：2025年2月20日 - 2025年3月20日
- **作品提交**：2025年4月1日 - 2025年4月30日
- **现场答辩**：2025年5月10日
- **结果公布**：2025年5月15日

## 🎯 赛题方向
1. **科学计算优化**：流体力学、分子动力学等领域的计算优化
2. **AI 训练加速**：深度学习模型训练的并行优化
3. **大数据处理**：大规模数据处理和分析的性能优化

## 🏅 奖项设置
- **金奖**：2名，奖金 8,000 元/队 + HPC 实训机会
- **银奖**：4名，奖金 5,000 元/队
- **铜奖**：6名，奖金 3,000 元/队
- **最佳性能奖**：2名，奖金 4,000 元/队
- **最佳创新奖**：2名，奖金 4,000 元/队

## 📋 参赛要求
- 参赛对象：东南大学全日制本科生、研究生
- 每队 1-3 名成员
- 必须基于 CANN 平台进行优化
- 提供完整的源代码和性能分析报告

## 🔬 技术要求

## 📊 评分标准
- **性能提升**（40%）：相比基准版本的加速比
- **技术创新性**（25%）：优化方法的创新性和技术难度
- **代码质量**（20%）：代码结构、注释、可维护性
- **文档完整性**（15%）：技术文档和演示效果
