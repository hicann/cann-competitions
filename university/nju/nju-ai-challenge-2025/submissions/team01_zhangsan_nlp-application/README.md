## 作品简介
本作品是一款基于 CANN 平台的智能问答教学系统，专为高校课程设计。系统能够理解课程相关问题，提供准确的答案和详细解释，并具备个性化学习路径推荐功能。系统采用了先进的 NLP 技术和知识图谱，支持多轮对话和复杂问题解答，有效提升了学习效率和教学质量。

## 技术架构

### 整体架构
```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   问题输入      │ ──> │  智能问答引擎   │ ──> │   答案生成      │
└─────────────────┘     └─────────────────┘     └─────────────────┘
        │                      ↑                       │
        │                      │                       │
        ▼                      │                       ▼
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   用户管理      │     │   知识图谱      │     │ 学习路径推荐    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
```

### 核心模块
1. **问题理解模块**：使用 BERT 模型进行意图识别和实体提取
2. **知识检索模块**：基于向量检索和知识图谱的混合检索
3. **答案生成模块**：结合检索结果和语言模型生成答案
4. **个性化推荐模块**：基于学习行为分析的个性化推荐

## 核心技术

### 1. 自然语言处理
- 采用 BERT-Base 模型进行文本编码
- 实现了意图识别、实体识别、关系抽取等功能
- 支持中文分词、词性标注、句法分析等基础 NLP 任务

### 2. 知识图谱技术
- 构建了课程领域知识图谱（包含 10,000+ 实体，20,000+ 关系）
- 实现了基于图数据库的高效查询
- 支持知识推理和图谱补全

### 3. 深度学习优化
- 模型基于 CANN 平台进行了深度优化
- 实现了模型量化和剪枝，提升推理速度
- 支持批处理和流式处理，满足不同场景需求

## CANN 特性应用

### 1. 模型部署优化
- 使用 CANN Model Converter 进行模型转换
- 利用 ATC 工具进行模型优化和编译
- 支持 INT8/FP16 混合精度推理

### 2. 并行计算加速
- 充分利用昇腾 NPU 的多核并行能力
- 实现了批处理并行和流水线并行
- 推理性能相比 CPU 提升了 8-10 倍

### 3. 内存优化
- 采用内存复用策略，降低内存占用
- 实现了高效的内存分配和释放
- 支持大模型的高效加载和推理

## 环境要求

### 硬件要求
- 昇腾 310/910 NPU 或兼容设备
- 至少 8GB 内存
- 推荐使用 NVIDIA GPU 进行训练

### 软件要求
- CANN 版本：6.0.0 及以上
- Python：3.7-3.9
- 依赖库：pytorch, transformers, jieba, neo4j

## 安装步骤

```bash
# 安装依赖
pip install -r requirements.txt

# 初始化知识图谱
python scripts/init_knowledge_graph.py --data_path=data/kg_data

# 部署模型到 CANN
bash scripts/deploy_model.sh --model_path=models/bert_base

# 启动服务
python app.py --host=0.0.0.0 --port=8000
```

## 使用方法

### API 接口
```bash
# 发送问题请求
curl -X POST http://localhost:8000/api/ask \
  -H "Content-Type: application/json" \
  -d '{"question": "什么是机器学习？", "user_id": "student_001", "course_id": "cs301"}'
```

### Web 界面
访问 http://localhost:8000 即可使用 Web 界面进行问答交互。

### 命令行工具
```bash
# 命令行交互
python cli.py --course=cs301 --user=student_001
```
