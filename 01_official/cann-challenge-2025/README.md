# CANN 全国挑战赛 2025

## 🏆 赛事介绍
CANN 全国挑战赛是 CANN 开源社区年度最高级别赛事，面向全国开发者，旨在推动昇腾 AI 生态建设，发掘优秀的 AI 应用和创新方案。

## 📅 赛事时间
- **报名时间**：2025年3月1日 - 2025年4月30日
- **作品提交**：2025年5月1日 - 2025年6月30日
- **评审时间**：2025年7月1日 - 2025年7月15日
- **结果公布**：2025年7月20日

## 🎯 赛题方向
1. **算子优化赛道**：针对特定算子进行性能优化
2. **模型部署赛道**：复杂模型的高效部署方案
3. **行业应用赛道**：基于 CANN 的行业创新应用

## 🏅 奖项设置
- **金奖**：3名，奖金 100,000 元/队
- **银奖**：6名，奖金 50,000 元/队
- **铜奖**：10名，奖金 20,000 元/队
- **最佳创新奖**：3名，奖金 30,000 元/队
- **最佳应用奖**：3名，奖金 30,000 元/队

## 📋 参赛要求
- 每队 1-5 名成员
- 必须使用 CANN 平台
- 作品必须为原创
- 提供完整的源代码和文档
