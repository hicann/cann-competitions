# 昇腾 CANN 算子测试赛 · 全国首届智能算子测试大赛

## 🏆 赛事介绍
本届大赛以"湾区智测·龙岗共建 - 昇腾 CANN 算子测试赛"为主题，由**深圳市龙岗区人工智能（机器人）署**指导，**全国大学生软件测试大赛组委会**、**CSDN**、**深圳河套学院**联合主办，**智子芯元**与**赛测科技**提供技术支持，面向全国高校在校生（专科、本科、研究生）开展。

大赛紧密围绕国家人工智能发展战略，聚焦昇腾 AI 基础软件生态核心"CANN 算子"，是一项集技术攻关、人才培养与生态建设于一体的专项活动。大赛旨在通过系统化和实战化的测试竞赛，选拔和培养一批深刻理解国产 AI 软硬件技术栈、具备智能算子质量工程能力的复合型测试人才。此次大赛的核心成果将陆续发布于 GitCode 平台，进一步推动人工智能开源生态发展。

## 📋 参赛要求

参赛者需针对组委会指定的 CANN 算子，在指定昇腾硬件环境中，设计并执行系统性测试。核心目标是发现、复现并分析算子在特定输入条件下可能存在的功能错误或数值精度偏差。这是从基础软件栈开发与质量保障的核心工程环节，直接关系到计算平台的可靠性与最终 AI 应用的准确性。竞赛模拟了 AI 芯片企业或大型科技公司中，测试开发工程师、软件质量工程师的真实工作场景。

大赛由高校的 1—3 名学生组队参赛，以团队负责人提交成果，共同完成竞赛任务。

## 🛠️ 赛前准备

参赛者需在规定时间内完成 CANN 环境的配置与试用，熟悉 CANN 算子的基本运行机制和测试接口，能够正确运行示例算子测试程序，并在测试实践过程中初步开展算子正确性与精度相关问题的识别与记录，按照要求尝试提交 Issue。该阶段不进行成绩评定，主要用于技术储备和参赛资格确认。主要目标是使参赛者熟悉昇腾开发环境，掌握 CANN 算子测试的基本流程与 Issue 提交规范。

预选赛采用 **CANN Simulator** 完成。CANN Simulator 是一款面向算子开发场景的 SoC 级芯片仿真工具，用于分析运行在 AI 仿真器上的 AI 任务在各阶段的精度和性能数据。该工具有助于用户进行深度性能调优，使研发人员在无法获取硬件芯片资源紧缺的情况下，也能获得与真实芯片几乎一致的验证效果和性能反馈。
